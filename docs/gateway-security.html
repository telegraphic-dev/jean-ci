Agent gateway security guide

Recommended binding

OPENCLAW_GATEWAY_URL=http://coolify-proxy/openclaw
OPENCLAW_GATEWAY_TOKEN=replace-me

Least privilege

  • Use a dedicated token for jean-ci.
  • Keep gateway access behind a private proxy or internal network path.
  • Do not expose the gateway directly on a public interface.

Rotation

  1. Create a new gateway token.
  2. Deploy jean-ci with the new token.
  3. Verify /api/health and a sample PR review.
  4. Revoke the old token.

Verification command:

curl -s -H "Authorization: Bearer $OPENCLAW_GATEWAY_TOKEN" "$OPENCLAW_GATEWAY_URL/health"
