Quickstart

Goal: install jean-ci and get one PR review running in under 10 minutes.

1. Start the app

cp .env.example .env
npm ci
npm run build
npm run start

Verification command:

curl -s http://localhost:3000/api/health

2. Add one review prompt

mkdir -p .jean-ci/pr-checks
cat > .jean-ci/pr-checks/security.md <<'EOF'
# Security Review

## Purpose
Catch blocking security regressions before merge.

## Review Instructions
Review the diff for exposed secrets, missing authorization checks, unsafe input handling, and injection risks.
Only fail for merge-blocking findings.

## Verdict Criteria
- **FAIL** if the diff introduces a real security vulnerability or exposes secrets.
- **PASS** if no blocking security issues are present in the diff.
EOF

Verification command:

git add .jean-ci/pr-checks/security.md && git commit -m "chore: add jean-ci security review"

3. Trigger a review

gh pr comment <pr-number> --body "/review"

Expected result: a global jean-ci / Code Review check, one check for security.md, and a PR review comment.
